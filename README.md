# ♻ EcoTrack — B2B E-Waste Disposal Management Platform

EcoTrack is a full-stack web application that helps businesses track, 
log, manage, and report electronic waste disposal in compliance with 
India's E-Waste Management Rules 2022.

## Tech Stack
- Frontend: React 19, Vite, Tailwind CSS, Privy Auth, Recharts
- Backend: Node.js, Express.js, MongoDB, Nodemailer, PDFKit
- Auth: Privy.io (passwordless email OTP)

## Features
- E-waste logging with category, weight, condition, vendor
- 5-step disposal lifecycle tracking with QR codes
- CO₂ impact calculator
- PDF & CSV audit report export
- Automated email reminders & monthly summaries
- Analytics charts and compliance score dashboard
